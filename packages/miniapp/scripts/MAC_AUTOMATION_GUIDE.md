# Mac 微信开发者工具 - 开启自动化方法

## 📍 Mac 版本自动化入口

Mac 微信开发者工具的自动化功能位置与 Windows 不同，请按以下步骤操作：

---

## ✅ 方法 1: 菜单栏 (最常用)

### 步骤 1: 打开菜单
点击顶部菜单栏 **"工具" (Tools)**

### 步骤 2: 找到自动化选项
在工具下拉菜单中找到:
```
工具 
  ├── 构建 npm
  ├── 运行 npm
  ├── 自动化(A)          ← 就是这个！
  ├── 编译模式
  ├── 刷新
  └── 设置...
```

如果没有看到 "自动化"，可能需要滚动菜单。

### 步骤 3: 点击自动化
点击后会打开自动化面板。

---

## ✅ 方法 2: 快捷键

### Mac 快捷键:
```
Cmd + Shift + A
```

如果快捷键不生效，可能与其他软件冲突。

---

## ✅ 方法 3: 左侧边栏图标

### 步骤 1: 查看左侧边栏
微信开发者工具窗口左侧有一列图标:
- 项目
- 模块
- 文件
- 安全
- **自动化** (机器人图标 🤖)  ← 就是这个！
- 云开发
- 控制台
- 帮助

### 步骤 2: 点击自动化图标
点击机器人图标 🤖 打开自动化面板。

---

## ✅ 方法 4: 状态栏按钮

在底部状态栏查找:
- 如果看到 "自动化" 字样按钮，点击它
- 或 "Auto" 按钮

---

## 🔍 如果找不到自动化选项?

### 检查 1: 版本问题
```
微信开发者工具 → 关于微信开发者工具
```
确保版本 >= 1.02.1803250

### 检查 2: 项目类型
自动化功能只对**小程序项目**可用，不支持公众号网页项目。

### 检查 3: 重新导入项目
如果自动化选项灰显:
1. 关闭项目
2. 重新导入项目
3. 等待编译完成
4. 再尝试打开自动化

---

## 🎯 自动化面板打开后的设置

### 开启自动化服务:

1. **打开自动化面板后**
2. **点击 "开启自动化" 或 "启用" 按钮**
3. **确认端口号**: `47748`

### 状态确认:
- ✅ 端口 47748 已监听
- ✅ "自动化" 按钮变为绿色/蓝色
- ✅ 显示 "已连接" 或类似状态

---

## 🔧 验证自动化是否开启

### 方法 A: 检查端口监听
```bash
lsof -i :47748
```
预期输出:
```
COMMAND   PID USER   FD   TYPE             DEVICE SIZE/OFF NODE NAME
wechatweb 57153  czh   58u  IPv4 0x31f77113d0b05e20      0t0  TCP localhost:47748 (LISTEN)
```

### 方法 B: 运行测试脚本
```bash
cd packages/miniapp
node scripts/ui-tester.js
```
如果看到:
```
✅ 连接成功!
🚀 小程序已启动
```
说明自动化已开启。

---

## ⚠️ 如果自动化选项灰显/不可点击

**原因**: 项目未完全编译或未在模拟器中加载

**解决**:
1. 确保模拟器中页面完全显示
2. 控制台无红色错误
3. 点击 "刷新" (Cmd + R)
4. 等待加载完成后再打开自动化

---
