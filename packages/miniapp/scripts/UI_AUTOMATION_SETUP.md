## 🔧 端口问题排查

### 问题 1: 端口 47748 未监听
```
❌ 错误: connect ECONNREFUSED 127.0.0.1:47748
```

**解决**:
1. 微信开发者工具 → 设置 → 安全设置
2. 勾选 "开启服务端口"
3. 确认端口 47748

---

### 问题 2: 自动化服务返回 404
```
❌ 错误: Unexpected server response: 404
```

**解决**:
1. 在开发者工具中: 工具 → 自动化
2. 点击 "开启自动化" 或 "启用自动化"

---

### 问题 3: WebSocket 连接失败
```
❌ 错误: connect ETIMEDOUT
```

**解决**:
1. 确保项目已在模拟器中加载
2. 确保页面完全渲染
3. 重启微信开发者工具

---

### 问题 4: 项目未编译
```
❌ 错误: 项目未加载
```

**解决**:
1. 确保 miniapp 项目已构建
```bash
cd packages/miniapp
npm run dev
```
2. 在开发者工具中刷新项目

## ⚠️ 注意事项

1. **自动化端口必须开启** - 否则无法连接
2. **项目必须加载** - 模拟器中必须显示页面
3. **页面必须完全渲染** - 等待 UI 加载完成
4. **不要关闭开发者工具** - 测试期间保持运行
5. **多个项目** - 确保测试的是正确项目
