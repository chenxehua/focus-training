```

    ↓
  测评ID: 16
  预计时长: 15分钟
  当前阶段: 问卷
```

### 5.2 随机问卷抽取
- **抽取数量**: 5题 (符合5-7题要求)
- **维度覆盖**: 5个维度
  - sustained_attention (持续注意力)
  - selective_attention (选择性注意力)
  - attention_shifting (转移注意力)
  - working_memory (工作记忆)
  - impulse_control (冲动控制)
- **验证**: 无重复题目 ✅
