## 一、项目概述

### 1.1 产品简介

- **产品名称**：专注星球（FocusKids）
- **目标用户**：4-12岁儿童及其家长
- **核心价值**：用App的价格，获得线下机构80%的训练效果
- **产品口号**：每天10分钟，专注伴成长

### 1.2 MVP阶段目标

- **核心游戏**：3个（舒尔特方格、听声辨数、图案记忆）
- **付费模式**：单一年度会员（199元/年）
- **DAU目标**：1000+
- **付费转化率**：8%+
- **次日留存**：40%+
- **开发周期**：12周
- **预算**：28万RMB

### 1.3 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                    表现层                                │
│  微信小程序 (primary) │ React Native iOS/Android (P2)   │
├─────────────────────────────────────────────────────────┤
│                    业务层                                │
│     Node.js / Express.js / Nest.js                      │
├─────────────────────────────────────────────────────────┤
│                    数据层                                │
│         MySQL 8.0 + Redis 6.0                           │
├─────────────────────────────────────────────────────────┤
│                    AI引擎层                              │
│    LLM (GPT-4/国产替代) + 推荐算法 + 自适应难度          │
└─────────────────────────────────────────────────────────┘
```
