## 统一规范

### 年龄分级
| 年龄组 | 级别标识 | 特点 |
|--------|----------|------|
| 4-6岁 | 幼儿组 | 简单规则、慢速度、多提示 |
| 7-9岁 | 小学组 | 中等难度、规则切换、适度挑战 |
| 10-12岁 | 高年级组 | 高难度、快速度、抗干扰 |

### 难度等级
每个游戏均设计L1-L4四级难度：
- **L1**：入门级
- **L2**：基础级
- **L3**：进阶级
- **L4**：挑战级（仅高年龄组开放）

### 数据格式
所有游戏统一使用JSON格式记录训练数据：
```json
{
  "game_id": "game_X",
  "session_id": "xxx",
  "age_group": "7-9岁",
  "records": [...],
  "summary": {...}
}
```
