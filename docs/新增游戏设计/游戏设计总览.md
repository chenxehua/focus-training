# 儿童专注力训练 - 游戏设计总览

## 一、项目概述

本项目为儿童专注力训练工具，包含**9款精心设计的注意力训练游戏**，覆盖4-12岁儿童的不同发展阶段，通过科学分级的游戏化训练提升儿童的专注力、记忆力、视觉追踪等核心认知能力。

### 项目信息
- **项目ID**：7634513359215329315
- **游戏总数**：9款
- **适龄范围**：4-12岁
- **合规定位**：专注力训练工具（非医疗/诊断产品）

## 三、完整游戏列表

### 3.1 已完成游戏（5款）

#### 游戏1：舒尔特方格
- **训练目标**：注意力的稳定性、广度、搜索效率
- **游戏机制**：在5×5-10×10的方格中按顺序点击数字
- **年龄分级**：4-6岁(L1-L3)、7-9岁(L1-L4)、10-12岁(L1-L5)
- **游戏化包装**："数字大闯关"主题
- **文档路径**：`./长期计划/儿童专注力训练/已完成游戏/`

#### 游戏2：听声辨数
- **训练目标**：听觉注意力、工作记忆、反应速度
- **游戏机制**：听数字音频，判断奇偶/大小关系
- **年龄分级**：4-6岁(3位)、7-9岁(4-5位)、10-12岁(5-6位)
- **游戏化包装**："小小音乐家"主题
- **文档路径**：`./长期计划/儿童专注力训练/已完成游戏/`

#### 游戏3：图案记忆
- **训练目标**：视觉记忆、工作记忆、模式识别
- **游戏机制**：显示图案序列，记忆后按顺序回忆
- **年龄分级**：4-6岁(3-4个)、7-9岁(5-7个)、10-12岁(8-10个)
- **游戏化包装**："记忆翻翻乐"主题
- **文档路径**：`./长期计划/儿童专注力训练/已完成游戏/`

#### 游戏4：视觉追踪
- **训练目标**：视觉追踪、持续注意力、抗干扰
- **游戏机制**：追踪屏幕上移动的指定目标
- **年龄分级**：4-6岁(1-2目标)、7-9岁(2-3目标)、10-12岁(3-4目标)
- **游戏化包装**："小小观察员"主题
- **文档路径**：`./长期计划/儿童专注力训练/已完成游戏/`

#### 游戏5：节奏点击
- **训练目标**：节奏感知、持续注意力、反应控制
- **游戏机制**：跟随节拍点击正确位置
- **年龄分级**：4-6岁(慢速)、7-9岁(中速)、10-12岁(快速)
- **游戏化包装**："节奏小达人"主题
- **文档路径**：`./长期计划/儿童专注力训练/已完成游戏/`

### 3.2 新增游戏（4款）

#### 游戏6：听觉记忆
- **训练目标**：听觉注意力、工作记忆、顺序/逆序记忆
- **游戏机制**：播放数字序列，按顺序或逆序回答
- **年龄分级**：4-6岁(3-5位)、7-9岁(5-8位)、10-12岁(8-10位)
- **游戏化包装**："小侦探听口令"主题
- **文档路径**：`./新增游戏设计/游戏6_听觉记忆_小侦探听口令.md`

#### 游戏7：迷宫寻路
- **训练目标**：持续注意力、视觉追踪、执行功能
- **游戏机制**：从简单迷宫到复杂迷宫寻路
- **年龄分级**：4-6岁(5×5)、7-9岁(8×8-11×11)、10-12岁(12×12-15×15)
- **游戏化包装**："小熊找蜂蜜"/"宇航员登月"主题
- **文档路径**：`./新增游戏设计/游戏7_迷宫寻路_小熊找蜂蜜.md`

#### 游戏8：快速分类
- **训练目标**：认知灵活性、注意力转移、分类能力
- **游戏机制**：按不同规则快速分类物品
- **年龄分级**：4-6岁(单维度)、7-9岁(双维度+切换)、10-12岁(多维度+干扰)
- **游戏化包装**："物品整理小能手"主题
- **文档路径**：`./新增游戏设计/游戏8_快速分类_物品整理小能手.md`

#### 游戏9：追踪目标
- **训练目标**：视觉注意力、抗干扰能力、选择性注意
- **游戏机制**：追踪移动目标或寻找隐藏目标
- **年龄分级**：4-6岁(3-5目标)、7-9岁(5-8目标)、10-12岁(8-15目标)
- **游戏化包装**："追踪小星星"/"寻找隐藏小动物"主题
- **文档路径**：`./新增游戏设计/游戏9_追踪目标_追踪小星星.md`

## 四、年龄适配设计总览

### 4.1 各年龄组游戏推荐

| 年龄组 | 推荐游戏 | 训练重点 |
|--------|----------|----------|
| **4-6岁** | 游戏1、2、3、5、6、7 | 基础注意力、简单记忆、初步视觉追踪 |
| **7-9岁** | 游戏1、2、3、4、5、6、7、8、9 | 全面训练、多任务处理、规则切换 |
| **10-12岁** | 游戏1、4、6、7、8、9 | 高阶追踪、复杂分类、快速反应 |

### 4.2 难度梯度说明

每个游戏都采用**4级难度梯度**设计：

| 等级 | 难度描述 | 适用年龄 |
|------|----------|----------|
| L1 | 入门级 | 初次接触该游戏类型 |
| L2 | 基础级 | 有一定经验 |
| L3 | 进阶级 | 能够熟练操作 |
| L4 | 挑战级 | 高阶玩家/10-12岁 |

## 六、成就系统统一框架

### 6.1 星级评价标准

| 星级 | 达成条件 | 显示文案风格 |
|------|----------|--------------|
| ⭐ | 基础完成 | "新手称号" |
| ⭐⭐ | 稳定表现 | "进阶称号" |
| ⭐⭐⭐ | 优秀表现 | "高手称号" |
| ⭐⭐⭐⭐ | 卓越表现 | "大师称号" |
| ⭐⭐⭐⭐⭐ | 连续卓越 | "传奇称号" |

### 6.2 通用成就类型

- **连续挑战**：连续完成N轮游戏
- **完美通关**：单轮0错误
- **速度之星**：平均反应时间低于阈值
- **坚持不懈**：累计完成一定时长
- **全能选手**：体验所有游戏类型
