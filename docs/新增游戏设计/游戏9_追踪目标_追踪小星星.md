# 游戏9：追踪目标 - 追踪小星星 / 寻找隐藏小动物

## 一、游戏概述

### 训练目标
- **核心能力**：视觉注意力、抗干扰能力
- **辅助能力**：视觉追踪、选择性注意、短期记忆

### 适龄范围
4-12岁儿童

### 游戏时长
单轮1.5-3分钟

---

## 二、游戏规则说明

### 主题变体

#### 主题A：追踪小星星（4-8岁推荐）
- 场景：夜空背景，多颗星星缓慢移动
- 目标：追踪指定的1-2颗星星，观察它们的变化
- 视觉风格：梦幻星空、柔和光芒
- 难度要素：星星数量、移动速度、闪烁干扰

#### 主题B：寻找隐藏小动物（8-12岁推荐）
- 场景：森林/海底/花园等自然场景
- 目标：在众多动物中找到指定的目标动物
- 视觉风格：自然生态、丰富细节
- 难度要素：动物数量、伪装程度、场景复杂度

### 基本玩法（星星追踪模式）
1. **目标指定**：屏幕上出现"追踪这颗星！"指示，指定目标星星
2. **移动追踪**：多颗星星同时移动，目标星星会变色/闪烁
3. **记忆挑战**：目标星星短暂隐藏后出现，需继续追踪
4. **终点判定**：目标星星到达终点区域，报告看到的变化次数
5. **结果评估**：比较追踪正确率与目标变化次数

### 基本玩法（动物寻找模式）
1. **任务展示**：显示要寻找的目标动物图片（如：找出所有蓝色的鸟）
2. **场景扫描**：大量动物随机分布在场地上
3. **快速识别**：找出符合条件的目标并点击
4. **限时挑战**：规定时间内找出所有目标
5. **干扰排除**：避免点击错误目标（扣分）

## 三、分年龄难度设计

### 4-6岁（幼儿组）

#### 追踪模式（星星主题）
| 等级 | 星星总数 | 目标数 | 移动速度 | 时间限制 |
|------|----------|--------|----------|----------|
| L1 | 3颗 | 1颗 | 慢速 | 30秒 |
| L2 | 4颗 | 1颗 | 中速 | 35秒 |
| L3 | 5颗 | 1-2颗 | 中速 | 40秒 |

**设计特点**：
- 目标星星有独特颜色（黄色）vs 普通星星（白色）
- 移动速度慢，轨迹平滑
- 场景简洁，无复杂背景干扰
- 提供"放大镜"帮助定位
- 正确追踪率高（70-80%可达成）

#### 寻找模式（动物主题）
| 等级 | 动物总数 | 目标数 | 干扰项 | 时间限制 |
|------|----------|--------|--------|----------|
| L1 | 6只 | 2只 | 2种 | 45秒 |
| L2 | 8只 | 3只 | 3种 | 50秒 |
| L3 | 10只 | 4只 | 4种 | 55秒 |

**设计特点**：
- 目标动物特征明显（颜色鲜明、位置显眼）
- 干扰项差异大（不同种类）
- 动物静止或轻微移动
- 点击正确率高
- 失败后给予目标位置提示

### 7-9岁（小学组）

#### 追踪模式
| 等级 | 星星总数 | 目标数 | 移动速度 | 干扰项 | 时间限制 |
|------|----------|--------|----------|--------|----------|
| L1 | 5颗 | 1颗 | 中速 | 闪烁 | 30秒 |
| L2 | 6颗 | 2颗 | 中速 | 闪烁+变色 | 35秒 |
| L3 | 7颗 | 2颗 | 快速 | 闪烁+变色+假消失 | 40秒 |
| L4 | 8颗 | 2-3颗 | 快速 | 全部干扰 | 45秒 |

**设计特点**：
- 目标星星只在开始时高亮，之后需要凭记忆追踪
- 增加闪烁干扰（星星闪烁而非消失）
- 增加短暂变色干扰（星星短暂变白）
- 目标可能短暂消失1-2秒
- 追踪轨迹可视化（可回看）

#### 寻找模式
| 等级 | 动物总数 | 目标数 | 干扰类型 | 时间限制 |
|------|----------|--------|----------|----------|
| L1 | 12只 | 4只 | 种类不同 | 40秒 |
| L2 | 15只 | 5只 | 同类不同特征 | 45秒 |
| L3 | 18只 | 6只 | 相似伪装 | 50秒 |
| L4 | 20只 | 7只 | 全部伪装 | 55秒 |

**设计特点**：
- 目标特征更细致（颜色+大小+数量要求）
- 干扰项更相似（需要仔细分辨）
- 部分动物有轻微移动
- 隐藏位置更多样
- 点击错误有惩罚机制

### 10-12岁（高年级组）

#### 追踪模式
| 等级 | 星星总数 | 目标数 | 移动速度 | 干扰类型 | 时间限制 |
|------|----------|--------|----------|----------|----------|
| L1 | 8颗 | 2颗 | 快速 | 变色+消失 | 35秒 |
| L2 | 10颗 | 3颗 | 快速 | 变色+消失+分身 | 40秒 |
| L3 | 12颗 | 3颗 | 极快 | 全部+轨迹混淆 | 45秒 |
| L4 | 15颗 | 4颗 | 极快 | 终极干扰 | 50秒 |

**干扰类型详解**：
- **闪烁**：星星随机闪烁（视觉干扰）
- **变色**：目标短暂失去颜色标记
- **消失**：目标短暂完全消失（1-3秒）
- **分身**：出现多个类似目标，只有一个是真目标
- **轨迹混淆**：留下假轨迹误导

#### 寻找模式
| 等级 | 动物总数 | 目标数 | 伪装程度 | 时间限制 |
|------|----------|--------|----------|----------|
| L1 | 20只 | 6只 | 40%伪装 | 45秒 |
| L2 | 25只 | 8只 | 60%伪装 | 50秒 |
| L3 | 30只 | 10只 | 80%伪装 | 55秒 |
| L4 | 35只 | 12只 | 极限伪装 | 60秒 |

**设计特点**：
- 目标与背景高度融合
- 需要组合条件筛选（颜色+形状+动作）
- 场景更加复杂（多层背景）
- 计时更严格
- 支持多轮连续挑战

## 四、UI交互设计要点

### 界面布局（星星追踪主题）

```
│      ★  ·    ·    ★             │
│        ↖   ↓    ↗              │

│          ↙  ↘                  │

│                                 │
├─────────────────────────────────┤

│   [✓ 报告完成]                 │

└─────────────────────────────────┘
```

### 交互流程（追踪模式）
1. **目标指定**：2秒内高亮指定目标，语音提示"追踪这颗星！"
2. **自由追踪**：目标失去高亮，开始追踪
3. **变化检测**：目标发生闪烁/变色/消失等变化时，点击报告按钮
4. **实时记录**：系统记录变化次数和时间点
5. **结果判定**：时间到后，与正确答案对比

### 交互流程（寻找模式）
1. **任务展示**：显示目标条件（"找出所有蓝色的鸟"）
2. **扫描时间**：5秒预览时间
3. **寻找开始**：倒计时开始，大量动物出现
4. **点击确认**：点击找到的目标
5. **结果评估**：正确率 + 完成时间

### 视觉设计
- **星星主题**：
  - 背景：深蓝渐变星空
  - 星星：金色/白色/彩色
  - 目标：黄色+光晕
  - 干扰：灰色/闪烁
- **动物主题**：
  - 背景：自然生态场景
  - 动物：卡通风格，辨识度高
  - 目标：边缘高亮
  - 干扰：部分透明/伪装色

## 七、技术实现建议

### 星星追踪技术
```javascript
// 星星对象结构
const star = {
  id: 'star_1',
  type: 'target' | 'decoy' | 'normal',
  color: '#FFD700',
  size: 20,
  position: { x: 100, y: 200 },
  velocity: { x: 2, y: 1 },
  state: 'visible' | 'flickering' | 'hidden' | 'color_changed',
  changeLog: [] // 记录变化时间点
};
```

### 移动算法
- 随机方向移动，碰到边界反弹
- 目标星星可能刻意接近边界增加追踪难度
- 分身模式：复制目标星星属性，位置随机

### 干扰事件系统
```javascript
// 干扰事件类型
const interferenceEvents = [
  { type: 'flicker', duration: 200, interval: 500 },
  { type: 'color_change', duration: 1000 },
  { type: 'disappear', duration: 1500 },
  { type: 'clone', count: 3, duration: 2000 }
];
```

### 动物场景生成
- 预设场景模板（森林、海底、花园等）
- 随机放置动物，确保位置不重叠
- 根据难度调整动物大小比例
- 目标动物位置伪随机但保证可达

## 八、变体设计建议

### 模式A：记忆追踪
- 目标星星消失后，需在星星重现时准确点击
- 训练工作记忆和预测能力

### 模式B：多点追踪
- 同时追踪多个目标（2-4个）
- 记录每个目标的追踪准确率

### 模式C：场景扫描
- 复杂场景中找指定物品
- 物品会短暂出现/消失
- 训练快速搜索能力

### 模式D：协作模式
- 两人合作追踪不同目标
- 交流信息协作完成任务

## 九、合规提示

⚠️ **重要提醒**：
- 星星闪烁频率适中，避免诱发光敏性问题
- 动物图片避免过于写实或恐怖
- 干扰设计应增加挑战性而非造成挫败感
- 界面元素移动速度可调节，保护儿童视觉
- 频繁失败时自动降低难度，体现正向引导
