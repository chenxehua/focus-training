## 1. 设计原则

### 1.1 核心设计原则

#### 儿童端设计原则

| 原则 | 说明 | 实施要点 |
|------|------|----------|
| **游戏化优先** | 将训练转化为游戏体验 | 角色、任务、奖励、反馈 |
| **正向反馈** | 强调成功而非失败 | 鼓励语、成功动画、进步提示 |
| **简洁直观** | 降低认知负担 | 大图标、少文字、单任务 |
| **趣味性** | 吸引儿童注意力 | 卡通形象、动态效果、音效 |
| **安全感** | 营造安心氛围 | 柔和色调、圆润设计、亲切语言 |

#### 家长端设计原则

| 原则 | 说明 | 实施要点 |
|------|------|----------|
| **数据驱动** | 用数据说话 | 可视化图表、趋势对比 |
| **效率优先** | 快速获取信息 | 卡片布局、重点突出 |
| **专业可信** | 建立专业形象 | 科学术语、专家内容 |
| **可控性** | 让家长掌控全局 | 设置便捷、权限明确 |
| **简洁清晰** | 信息不过载 | 分层展示、适当留白 |

### 1.2 双端设计差异

```
┌─────────────────────────────────────────────────────────────┐
│                      双端设计差异                           │
├─────────────────────────┬─────────────────────────────────┤
│       儿童端             │           家长端                │
├─────────────────────────┼─────────────────────────────────┤
│  风格：活泼、卡通、童趣   │  风格：简洁、专业、清晰          │
│  色调：明亮、柔和        │  色调：沉稳、品牌色              │
│  交互：游戏化、大触摸    │  交互：传统、功能导向           │
│  信息：简化、可视化      │  信息：完整、专业               │
│  激励：即时、频繁       │  激励：成就、数据对比           │
└─────────────────────────┴─────────────────────────────────┘
```

### 1.3 无障碍设计

| 要求 | 说明 |
|------|------|
| 色彩对比 | 文本与背景对比度≥4.5:1 |
| 触摸目标 | 最小点击区域48×48dp |
| 字体大小 | 正文≥14sp，标题≥18sp |
| 操作反馈 | 明确的状态反馈 |
| 文字辅助 | 图片配文字说明 |

## 2. 色彩与风格定义

### 2.1 色彩系统

#### 主色彩（Brand Color）

| 用途 | 色值 | 说明 |
|------|------|------|
| **主色** | #6C63FF | 品牌识别色，用于主要按钮、强调元素 |
| **主色浅** | #8B85FF | 背景、hover状态 |
| **主色深** | #5046E5 | 按下状态、阴影 |

#### 辅助色（Secondary Colors）

| 用途 | 色值 | 说明 |
|------|------|------|
| **成功** | #4CAF50 | 正确反馈、成就徽章 |
| **警告** | #FFC107 | 提醒、中等难度 |
| **错误** | #F44336 | 错误反馈（儿童端少用） |
| **信息** | #2196F3 | 提示信息 |

#### 儿童端专属色（活泼版）

| 用途 | 色值 | 说明 |
|------|------|------|
| **阳光黄** | #FFD93D | 星星、金币、高光 |
| **天空蓝** | #6EC6FF | 背景、装饰 |
| **草地绿** | #6BCB77 | 成功、完成 |
| **棉花糖粉** | #FFB5C5 | 女生喜好、温馨氛围 |
| **橙子橙** | #FF9F45 | 能量、激励 |

#### 家长端专属色（专业版）

| 用途 | 色值 | 说明 |
|------|------|------|
| **深蓝** | #2C3E50 | 标题、重点文字 |
| **灰黑** | #34495E | 正文文字 |
| **浅灰** | #ECF0F1 | 背景、分割线 |
| **白色** | #FFFFFF | 卡片背景 |

### 2.2 色彩应用规范

#### 儿童端配色方案

```
页面背景：渐变蓝（#6EC6FF → #8B85FF）
卡片背景：白色 #FFFFFF，圆角16dp
主要按钮：主色 #6C63FF
成功反馈：草地绿 #6BCB77
激励元素：阳光黄 #FFD93D
错误提示：柔和红（少用）#FF8A80
```

#### 家长端配色方案

```
页面背景：浅灰 #F5F7FA
卡片背景：白色 #FFFFFF，圆角12dp
主要按钮：主色 #6C63FF
标题文字：深蓝 #2C3E50
正文文字：灰黑 #34495E
分割线：浅灰 #ECF0F1
```

### 2.3 字体系统

#### 字体规范

| 类型 | 字体 | 字号 | 字重 | 使用场景 |
|------|------|------|------|----------|
| 标题1 | 系统字体 | 24sp | Bold | 页面大标题 |
| 标题2 | 系统字体 | 20sp | SemiBold | 区块标题 |
| 标题3 | 系统字体 | 18sp | Medium | 卡片标题 |
| 正文 | 系统字体 | 16sp | Regular | 主要内容 |
| 说明 | 系统字体 | 14sp | Regular | 辅助说明 |
| 标签 | 系统字体 | 12sp | Medium | 标签、小按钮 |

#### 儿童端字体特点

- 圆体字形，友好亲切
- 字号偏大，便于阅读
- 可配合卡通字体展示成就

### 2.4 风格定义

#### 整体风格

| 属性 | 儿童端 | 家长端 |
|------|--------|--------|
| **风格** | 活泼童趣 | 简洁专业 |
| **圆角** | 大圆角（16-24dp） | 中圆角（8-12dp） |
| **阴影** | 柔和阴影 | 轻微阴影 |
| **图标** | 卡通风格 | 线性图标 |
| **插图** | 可爱卡通 | 简约图形 |

#### 视觉层次

```
第一层：品牌色 #6C63FF
第二层：辅助色（绿/黄/蓝）
第三层：中性色（灰/白）
第四层：背景色（渐变/纯色）
```

## 6. 适配要求

### 6.1 屏幕适配

#### 适配策略

| 策略 | 说明 | 适用场景 |
|------|------|----------|
| 响应式布局 | 弹性尺寸适配 | 主要页面 |
| 固定高宽比 | 保持比例缩放 | 游戏区域 |
| 安全区域适配 | 适配刘海屏等 | 全局 |

#### 尺寸规范

| 设备类型 | 屏幕宽度 | 设计基准 |
|----------|----------|----------|
| 小屏手机 | 320-360dp | 最小适配 |
| 标准手机 | 360-400dp | 主要适配 |
| 大屏手机 | 400-440dp | Plus/Max |
| 平板 | 600dp+ | 横屏优先 |

### 6.2 系统适配

#### 微信版本

| 版本 | 支持要求 |
|------|----------|
| 微信 7.0+ | 最低支持 |
| 微信 8.0+ | 推荐支持 |
| 微信 8.0+ | 完全适配 |

#### 操作系统

| 系统 | 最低版本 | 推荐版本 |
|------|----------|----------|
| iOS | 12.0 | 15.0+ |
| Android | 8.0 | 11.0+ |

### 6.3 深色模式

| 模式 | 是否支持 | 说明 |
|------|----------|------|
| 浅色模式 | ✅ 强制 | 儿童端始终浅色 |
| 深色模式 | ⚠️ 家长端可选 | 家长端可选 |

> 注：儿童端不建议支持深色模式，因为儿童用户的视觉发展需要明亮的色彩刺激，且深色模式可能影响专注力训练的效果呈现。
