## 1. 文档概述

### 1.1 文档目的

本PRD旨在为儿童专注力训练工具「专注星球」提供完整的产品需求规格说明，作为产品、开发、设计、测试团队的工作依据。文档涵盖产品愿景、功能需求、非功能需求、数据指标、技术架构、上线计划及风险评估等核心内容，确保各团队对产品有统一理解。

### 1.2 产品名称与Slogan

| 产品名称 | **专注星球**（FocusKids） |

### 1.3 产品愿景与使命

**愿景（3-5年）**：
- 成为国内最大的儿童专注力训练平台，覆盖100万+家庭
- 构建「AI+专注力」教育生态，成为家校协同的专注力提升基础设施

**使命**：
用科技的力量，让每个孩子都能获得科学、便捷、有效的专注力训练。

**核心价值主张**：

| 角色 | 价值点 |
|------|--------|
| **对孩子** | 🎮 好玩（游戏化训练）+ 🏆 成就感（即时反馈）+ 🤝 陪伴（AI小助手） |
| **对家长** | 📊 看得见（三层效果可视化）+ 📚 学得到（家长学院）+ 💰 值得（199元起） |
| **对学校** | 📋 减负担（全校专注力普测）+ 📈 可追踪（班级学情看板）+ 🔗 可协同（家校互通） |

### 1.5 术语定义

| 术语 | 定义 |
|------|------|
| **专注力** | 注意力（Attention），特指儿童维持对任务的指向和集中的心理活动能力 |
| **ADHD** | 注意力缺陷多动障碍，本产品定位为训练工具，不具备诊断资质 |
| **ZPD** | 最近发展区（Zone of Proximal Development），指儿童现有水平与潜在发展水平之间的区域 |
| **CPT** | 持续操作测试（Continuous Performance Test），评估持续注意力的金标准工具 |
| **MVP** | 最小可行产品（Minimum Viable Product） |
| **PMF** | 产品市场契合（Product-Market Fit） |
| **DAU** | 日活跃用户数（Daily Active Users） |
| **MAU** | 月活跃用户数（Monthly Active Users） |
| **LTV** | 用户生命周期价值（Lifetime Value） |
| **CAC** | 用户获取成本（Customer Acquisition Cost） |

## 2. 产品背景与定位

### 2.1 市场背景（基于竞品调研与问题分析报告）

#### 2.1.1 市场规模与增长趋势

| 市场指标 | 数据 | 说明 |
|----------|------|------|
| 市场规模（2025E） | 167-280亿元 | 项目建设方案引用数据 |
| 年复合增长率 | 17%-25% | 市场预测 |
| 目标用户规模 | 1.2亿（3-12岁儿童） | 中国儿童人口基数 |
| 市场渗透率 | 5%-8% | 说明绝大多数需求未被满足 |
| 专注力训练设备出货量 | 2023年42万台，同比+168% | 艾瑞咨询报告 |

**关键洞察**：市场渗透率仅5%-8%，意味着巨大的增长空间。

#### 2.1.2 用户痛点分析

| 痛点类型 | 数据表现 | 占比 |
|----------|----------|------|
| 家长因孩子注意力不集中崩溃 | 育儿焦虑严重 | 52.32% |
| 学龄儿童专注力不达标 | 普遍性问题 | 38.5% |
| 存在上课走神、作业拖拉问题 | 6-12岁儿童 | 每3人中1人 |
| 线下机构价格高（年费1.5-5万） | 价格门槛高 | - |
| 现有App专业性不足 | 多为小游戏堆砌 | 普遍现象 |
| 家长无法追踪训练效果 | 数据缺失 | 普遍现象 |

#### 2.1.3 竞品优劣势分析

| 竞品类型 | 代表产品 | 优势 | 劣势 |
|----------|----------|------|------|
| 线下机构 | 各类训练中心 | 专业性强 | 价格高、时间成本高 |
| 舒尔特方格类App | 各种舒尔特App | 功能单一 | 无家长端、成人化设计 |
| 脑力训练类App | Lumosity, Peak | 科学背书强 | 成人定位、无家长端 |
| 综合教育类App | 斑马AI课 | 品牌大 | 专注力非核心 |
| ADHD辅助类App | ChomChom, JoonApp | 儿童友好 | 国内不可用 |

**市场空白**：
- 国内缺乏「科学背书+游戏化+家长端闭环」三位一体的专注力训练产品
- 价格区间200-500元/年存在巨大空白

#### 2.1.4 市场空白与机会

| 价格区间 | 定位 | 痛点 | 我们的机会 |
|----------|------|------|-----------|
| 0-200元/年 | 低端预算型 | 专业性不足、效果不明显 | 基础版定位 |
| **200-500元/年** | **中端性价比型** | **巨大空白** | **核心定位** |
| 5000-15000元/疗程 | 中端专业型 | 时间成本高 | - |
| 15000-50000元/疗程 | 高端定制型 | 价格过高 | - |

##### 4.1.1 手机号+验证码登录

| 属性 | 说明 |
|------|------|
| 优先级 | P0 |

| 属性 | 说明 |
|------|------|

|------|------|
