# 微信小程序页面清单与操作路径

## 一、总体概览

### 1.1 小程序信息
- **名称**：专注星球 (FocusKids)
- **定位**：面向4-12岁儿童的专注力训练应用
- **技术栈**：uni-app (Vue3 + TypeScript + Pinia)

### 1.2 页面总数
共 **29个页面**，分为以下6大模块：

| 模块 | 页面数 | 说明 |
|------|--------|------|
| 训练模块 | 11个 | 首页、游戏广场、9款训练游戏 |
| 家长报告模块 | 1个 | 周报查看与分析 |
| 个人中心模块 | 5个 | 登录、我的、成就、会员、评估 |
| 家长学院模块 | 5个 | 学院、文章、问答、提问 |
| 学校管理模块 | 4个 | 仪表盘、教师、班级、学生管理 |
| 训练推荐模块 | 1个 | 个性化推荐与周计划 |
| 登录模块 | 1个 | 手机+验证码/微信登录 |

### 1.3 底部导航栏（TabBar）
```
┌─────────┬─────────┬─────────┐
│ 训练首页 │ 家长报告 │   我的   │
│  (首页)  │  (报告)  │  (Profile) │
└─────────┴─────────┴─────────┘
```

---

## 二、TabBar页面（底部导航）

### 2.1 训练首页 (pages/index/index)

**功能说明**：
- 显示当前选中儿童的每日训练进度
- 展示推荐游戏卡片（最多4个）
- 显示今日打卡状态和连续打卡天数
- 查看今日训练记录列表

**操作路径**：
```
小程序启动 → 自动进入训练首页（TabBar第1个）
```

**关键交互**：
- 点击游戏卡片 → 进入对应游戏页面
- 点击「查看全部」→ 进入游戏广场
- 未添加孩子时显示引导提示

**页面元素**：
| 元素 | 说明 |
|------|------|
| 儿童头像 | 显示当前选中儿童的头像 |
| 打卡进度条 | 显示今日训练时长（目标15分钟） |
| 连续打卡徽章 | 显示持续训练天数 |
| 推荐游戏网格 | 2列布局展示游戏卡片 |
| 今日记录列表 | 显示最近5条训练记录 |

---

### 2.2 家长报告 (pages/parent/index)

**功能说明**：
- 查看选定儿童的周训练报告
- 展示本周训练概览（次数、时长、平均专注得分）
- 显示本周每日专注力趋势柱状图
- 展示游戏训练分布和成就亮点

**操作路径**：
```
底部TabBar → 点击「家长报告」
```

**关键交互**：
- 左右滑动切换不同儿童
- 报告自动加载当前儿童的本周数据

**页面元素**：
| 元素 | 说明 |
|------|------|
| 儿童切换器 | 横向滚动显示多个儿童头像 |
| 本周概览卡片 | 显示训练次数、总时长、平均得分 |
| 专注趋势图 | 7天柱状图展示每日专注得分 |
| 游戏分布 | 各游戏训练次数和平均得分 |
| 成就亮点 | 本周获得的成就列表 |

---

### 2.3 我的 (pages/profile/index)

**功能说明**：
- 用户登录/个人信息管理
- 儿童信息管理（添加/切换/查看详情）
- 访问会员中心、消息通知、联系客服、关于我们
- 退出登录

**操作路径**：
```
底部TabBar → 点击「我的」
```

**关键交互**：
- 点击登录区域 → 微信一键登录或手机验证码登录
- 点击「+ 添加」→ 弹出添加孩子表单
- 点击儿童卡片 → 切换为当前儿童
- 点击功能菜单项 → 跳转对应页面

**页面元素**：
| 元素 | 说明 |
|------|------|
| 用户头像 | 点击可获取微信头像 |
| 儿童列表 | 显示已添加的儿童卡片 |
| 功能菜单 | 消息通知、会员中心、联系客服、关于我们 |
| 退出登录 | 退出当前账号 |

## 三、训练模块

### 3.1 游戏广场 (pages/games/index)

**功能说明**：
- 展示全部9款训练游戏
- 按游戏类型（注意力、记忆、反应等）筛选
- 显示游戏锁定状态（会员专享）

**操作路径**：
```
训练首页 → 点击「查看全部」→ 进入游戏广场
```

**或通过**：
```
我的 → 找到「游戏广场」入口（如有）
```

**游戏分类**：
| 分类标签 | 说明 |
|----------|------|
| 全部 | 显示所有游戏 |
| 注意力 | 舒尔特方格、视觉追踪等 |
| 记忆 | 图案记忆、听觉记忆等 |
| 反应 | 反应速度、节奏点击等 |
| 感知 | 听声辨数、迷宫寻路等 |
| 冥想 | 呼吸训练等 |
| 观察 | 找不同等 |
| 计算 | 快速分类等 |
| 听觉 | 听声辨数、听觉记忆等 |

**关键交互**：
- 点击分类标签 → 筛选对应类型游戏
- 点击游戏卡片（已解锁）→ 进入游戏
- 点击游戏卡片（锁定）→ 提示升级会员并跳转会员页

---

### 3.2 舒尔特方格 (pages/game-schulte/index)

**功能说明**：
- 经典注意力训练游戏
- 在N×N方格中按顺序点击数字

**操作路径**：
```
游戏广场 → 选择「舒尔特方格」游戏卡片
```

**或**：
```
训练首页 → 推荐游戏 → 舒尔特方格
```

**游戏配置**：
- 难度等级：3×3, 4×4, 5×5, 6×6方格
- 训练维度：持续注意力、视觉搜索

## 五、个人中心模块

### 5.1 登录页面 (pages/login/index)

**功能说明**：
- 手机号+验证码登录
- 微信一键登录
- 用户协议确认

**操作路径**：
```
训练首页/家长报告/我的 → 未登录状态 → 点击登录区域
```

**关键交互**：
1. **手机登录**：
   - 输入手机号 → 点击「获取验证码」→ 输入验证码 → 点击「登录」

2. **微信登录**：
   - 点击「微信一键登录」→ 授权 → 自动登录

3. **登录成功**：
   - 自动跳转到首页

---

**套餐类型**：

| 年卡 | ¥xx | 12个月会员（最划算） |
| 永久卡 | ¥xx | 永久会员 |

**关键交互**：
- 选择套餐 → 确认支付 → 微信支付 → 开通成功

## 十、页面操作指南

### 10.1 首次使用流程

```
1. 打开小程序 → 进入训练首页（未登录）
         ↓
2. 点击登录区域 → 进入登录页面
         ↓
3. 选择登录方式：
   · 微信一键登录（推荐）
   · 手机号+验证码登录
         ↓
4. 登录成功后 → 返回训练首页
         ↓
5. 点击「+ 添加」→ 填写孩子信息（姓名、年龄、性别）
         ↓
6. 添加成功 → 自动选为当前孩子 → 可以开始训练
```

### 10.2 日常使用流程

```
1. 打开小程序 → 进入训练首页
         ↓
2. 确认当前孩子（如需切换，点击孩子卡片切换）
         ↓
3. 开始训练：
   · 直接点击首页推荐游戏
   · 点击「查看全部」进入游戏广场选择
         ↓
4. 完成游戏 → 自动记录成绩 → 可查看训练报告
         ↓
5. 查看报告：
   · 点击「家长报告」TabBar查看周报
   · 点击「成就中心」查看获得成就
   · 点击「专注力评估」查看详细评估
```

### 10.3 会员升级流程

```
1. 进入游戏广场 → 点击锁定游戏
         ↓
2. 提示「升级会员解锁更多游戏」
         ↓
3. 点击确定 → 进入会员中心
         ↓
4. 选择套餐（月卡/季卡/年卡/永久卡）
         ↓
5. 点击「立即开通」→ 确认支付
         ↓
6. 完成微信支付 → 开通成功 → 解锁全部游戏
```

## 十一、页面状态说明

### 11.1 登录状态判断

| 状态 | 显示内容 | 可用功能 |
|------|----------|----------|
| 未登录 | 登录引导区域 | 仅浏览公开内容，无法训练 |
| 已登录-无孩子 | 添加孩子引导 | 可登录，无法训练 |
| 已登录-有孩子 | 儿童卡片列表 | 可使用全部训练功能 |

### 11.2 会员状态判断

| 状态 | 游戏显示 | 功能限制 |
|------|----------|----------|
| 非会员 | 免费游戏可玩，付费游戏显示锁定 | 无法使用全部9款游戏 |
| 会员有效 | 所有游戏可玩 | 可使用全部功能 |
| 会员到期 | 到期后自动降级 | 无法使用会员专属功能 |

### 11.3 空状态处理

| 页面 | 空状态提示 | 引导操作 |
|------|------------|----------|
| 训练首页 | "还没有添加孩子信息" | 前往「我的」页面添加孩子 |
| 家长报告 | "本周暂无训练数据" | 完成游戏后查看专注力报告 |
| 成就中心 | "还没有任何成就" | 完成训练任务解锁成就 |
| 专注力评估 | "暂无评估数据" | 完成足够训练后自动生成 |
| 训练推荐 | "完成更多训练解锁个性化推荐" | 继续完成训练 |
