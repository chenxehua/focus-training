# 儿童专注力训练数据模型设计完整方案

> 版本：v1.0  
> 更新日期：2025年5月  
> 文档性质：产品技术开发指导文件

---

## 一、概述

本文档为儿童专注力训练产品提供完整的数据模型设计、数据分析框架和运营指导方案。设计方案遵循以下核心原则：

1. **隐私保护优先**：严格遵循《个人信息保护法》《儿童个人信息网络保护规定》，所有数据采集需获得监护人明确授权
2. **科学循证**：所有专注力指标基于认知心理学和神经科学研究成果，具有可量化、可验证性
3. **可落地执行**：方案可直接指导产品开发和数据分析工作

## 二、数据模型设计

### 2.1 设计原则

| 原则 | 说明 |
|------|------|
| 最小化原则 | 仅采集实现产品功能必需的数据字段 |
| 目的明确原则 | 每个字段需明确业务用途，禁止无明确目的的字段 |
| 分级存储原则 | 热数据、温数据、冷数据分层存储，定期归档 |
| 脱敏处理原则 | 原始用户数据不出生产环境，分析使用聚合/脱敏数据 |

### 4.1 儿童维度

#### 4.1.1 年龄分层分析

| 年龄段 | 心理发展特征 | 训练适配策略 | 核心关注指标 |
|--------|--------------|--------------|--------------|
| 3-5岁 | 注意力持续时间短（5-15分钟），以无意注意为主 | 游戏化程度高，单次时长5-10分钟，趣味性强 | 任务完成率、兴趣维持度 |
| 6-8岁 | 注意力持续时间延长（15-20分钟），有意注意开始发展 | 适度挑战，引入竞技元素，奖励机制 | 准确率、反应时、错误类型分布 |
| 9-12岁 | 注意力稳定性提升，可维持30分钟以上 | 可引入元认知训练，时间管理概念 | 综合得分、难度适应性、自我调节能力 |

#### 4.1.2 训练频次与时长分析

```
训练频次评估标准（按年龄）
┌─────────────────────────────────────────────────────────┐
│ 年龄段    │  推荐频次   │ 推荐单次时长  │  周总量上限  │
```

#### 4.1.3 各专注力指标变化追踪

**指标计算公式**

| 指标名称 | 计算公式 | 说明 |
|----------|----------|------|
| 持续注意力得分 | 正确反应数 × 100 / 总刺激数 - 惩罚分 | 惩罚分=错误数×5 |
| 选择性注意指数 | 正确识别目标数 / 总目标数 × 100 | 排除干扰正确率 |
| 分配注意效率 | (任务A正确率 + 任务B正确率) / 2 × 任务完成率 | 双任务协调能力 |
| 注意力转移速度 | 任务切换后首个正确反应的反应时 | 越小越好 |
| 反应时变异系数 | 标准差 / 平均反应时 | 稳定性指标，越小越稳定 |
| 冲动控制指数 | No-Go试次正确拒绝率 | 抑制优势反应能力 |

```python
def identify_golden_hours(child_id, lookback_days=30):
    """
    识别个体最佳训练时间段
    """
    # 1. 获取历史训练数据
    sessions = get_sessions(child_id, last_n_days=lookback_days)
    
    # 2. 按小时聚合表现数据
    hourly_stats = {}
    for session in sessions:
        hour = session.start_time.hour
        if hour not in hourly_stats:
            hourly_stats[hour] = {'scores': [], 'duration': 0}
        hourly_stats[hour]['scores'].append(session.score)
        hourly_stats[hour]['duration'] += session.duration
    
    # 3. 计算各时段综合得分
    hourly_scores = []
    for hour, stats in hourly_stats.items():
        avg_score = mean(stats['scores'])
        # 加权：表现占70%，时长稳定性占30%
        duration_score = min(stats['duration'] / 60, 1.0) * 100
        composite = avg_score * 0.7 + duration_score * 0.3
        hourly_scores.append({'hour': hour, 'score': composite})
    
    # 4. 排序并返回Top3
    sorted_hours = sorted(hourly_scores, key=lambda x: x['score'], reverse=True)
    return sorted_hours[:3]
```

```
def personalized_recommendation(child_profile):
    """
    基于协同过滤 + 内容推荐混合算法
    """
    # 1. 短板识别
    weak_dimensions = identify_weak_dimensions(child_profile)
    
    # 2. 候选游戏生成
    candidate_games = []
    for dim in weak_dimensions:
        games_for_dim = get_games_by_dimension(dim)
        candidate_games.extend(games_for_dim)
    
    # 3. 过滤已熟练游戏
    candidate_games = [
        g for g in candidate_games 
        if not is_mastered(g, child_profile)
    ]
    
    # 4. 协同过滤优化（找相似用户的最优选择）
    similar_users = find_similar_users(child_profile)
    cf_recommendations = get_popular_among_similar_users(similar_users)
    
    # 5. 混合排序
    final_games = hybrid_ranking(
        candidate_games,
        cf_recommendations,
        child_profile.preferences,
        child_profile.current_state
    )
    
    # 6. 生成训练计划
    return generate_training_plan(final_games, child_profile)

def identify_weak_dimensions(profile):
    """
    识别短板维度
    """
    # 获取同龄常模
    age_norm = get_age_norm(profile.age_group)
    
    # 计算各维度与常模的差距
    gaps = {}
    for dim, score in profile.dimension_scores.items():
        norm_score = age_norm[dim]
        gaps[dim] = (norm_score - score) / norm_score  # 差距百分比
    
    # 按短板程度排序
    sorted_gaps = sorted(gaps.items(), key=lambda x: x[1], reverse=True)
    
    # 返回Top3短板
    return [dim for dim, gap in sorted_gaps[:3]]
```

## 七、数据安全与隐私保护

### 7.1 合规框架

| 法规/标准 | 核心要求 | 产品落地措施 |
|-----------|----------|--------------|
| 《个人信息保护法》 | 最小必要、告知同意、安全保障 | 仅采集功能必需字段；首次注册明确告知并获同意 |
| 《儿童个人信息网络保护规定》 | 14岁以下需监护人同意 | 儿童账号由家长注册；数据处理需监护人授权 |
| 《个人信息安全规范》 | 分类分级、加密存储 | 儿童数据加密；按敏感程度分级管理 |
| GDPR（可选遵循） | 数据主体权利保障 | 提供数据导出、删除功能 |

### 7.3 隐私保护设计原则

| 原则 | 具体措施 |
|------|----------|
| **数据最小化** | 不采集与训练无关的信息；不强制收集真实姓名 |
| **目的限制** | 数据仅用于产品功能优化，不得用于他途 |
| **存储期限** | 儿童数据在账号注销后90天内删除 |
| **知情同意** | 隐私政策使用儿童友好语言；监护人确认二次验证 |
| **访问控制** | 员工访问需审批；日志全程记录 |
| **用户权利** | 支持数据导出、账号注销、授权撤回 |

| 年龄段 | 持续注意(秒) | 选择性注意(正确率) | 反应时(ms) | 工作记忆广度 |

*注：以上为参考范围，实际评估需结合多维度综合判断*

### 8.2 术语表

| 术语 | 定义 |
|------|------|
| 持续注意 | Sustained Attention，维持注意于连续性任务的能力 |
| 选择性注意 | Selective Attention，从干扰中提取目标信息的能力 |
| 分配注意 | Divided Attention，同时注意多个任务/信息源的能力 |
| 转移注意 | Alternating Attention，灵活切换注意焦点的能力 |
| 工作记忆 | Working Memory，暂时存储和操作信息的认知系统 |
| 冲动控制 | Impulse Control，抑制不当优势反应的能力 |
| Cohen's d | 效应量指标，用于衡量组间差异的实际大小 |
| 百分位 | Percentile，同龄群体中的相对位置 |
