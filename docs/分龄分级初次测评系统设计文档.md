# 分龄分级专注力初次测评系统设计文档

> **文档版本**：V1.1（精简版）  
> **编制日期**：2025年5月  
> **修订说明**：优化为5-10分钟快速测评，减少题目数量，提升用户体验  
> **适用范围**：产品、开发、测试团队使用  
> **产品名称**：专注星球（FocusKids）

---

## 设计理念

### 核心理念：快速、精准、友好

1. **快速完成**：总测评时长控制在8-10分钟内
2. **题库抽题**：80道题库，每次随机抽取5-7道
3. **精准评估**：每维度1-2道题即可评估

### 时间分配

| 环节 | 时长 | 说明 |
|------|------|------|
| 基础信息 | 1分钟 | 姓名(可选)、年龄、性别 |
| 问卷测评 | 2-3分钟 | 题库随机抽取5-7题 |
| 游戏测评 | 4-5分钟 | 2款适配游戏 |
| 报告生成 | 即时 | 实时生成 |

### 设计原则

1. **题库庞大**：设计80道题，覆盖7个评估维度
2. **智能抽取**：每次测评随机抽取5-7道题
3. **维度均衡**：确保每维度至少抽取1道题
4. **语言简化**：儿童友好，避免复杂表述
5. **游戏轻量**：每款游戏控制在1-2分钟内
6. **游戏替代部分问卷题**：减少家长负担

## 文档概述

### 1.1 文档目的

本文档详细描述专注星球儿童专注力初次测评系统的设计，包括需求分析、开发设计、数据库设计以及任务规划和拆分。该系统针对4-12岁儿童，根据不同年龄阶段设计个性化的测评问卷和游戏难度，确保测评结果科学、准确、有效。

### 1.2 产品背景

- **产品名称**：专注星球（FocusKids）
- **目标用户**：4-12岁儿童及其家长
- **核心价值**：用App的价格，获得线下机构80%的效果
- **产品Slogan**：每天10分钟，专注伴成长
- **测评时长**：5-10分钟快速测评
- **测评题量**：14道问卷 + 2款游戏

### 1.3 术语定义

| 术语 | 定义 |
|------|------|
| 初次测评 | 儿童首次使用产品时进行的全面专注力评估 |
| 分龄分级 | 根据儿童年龄分组，并为每组设计不同难度的测评内容 |
| 百分位排名 | 将儿童测评结果与同龄群体进行比较得出的相对位置 |
| 7维度评估 | 持续注意力、选择性注意力、分配注意力、转移注意力、工作记忆、冲动控制、反应速度 |
| ZPD | 最近发展区（Zone of Proximal Development），指儿童现有水平与潜在发展水平之间的区域 |

### 2. 年龄分组设计

#### 2.1 年龄分组标准

| 年龄组 | 年龄范围 | 认知发展特点 | 测评设计原则 |
|--------|----------|--------------|--------------|
| **幼儿组** | 4-5岁 | 语言快速发展，好奇心强，注意力以无意注意为主 | 简化语言，减少题目，延长展示时间 |
| **学前期** | 6-7岁 | 自我控制能力增强，开始形成有意注意 | 中等难度，适当延长单次展示时间 |
| **小学低年级** | 8-9岁 | 执行功能快速发展，学业任务繁重 | 较高难度，符合学校要求 |
| **小学高年级** | 10-12岁 | 认知能力接近成人，可接受复杂任务 | 高难度，接近成人标准 |

#### 2.2 年龄组详细参数

| 参数 | 4-5岁 | 6-7岁 | 8-9岁 | 10-12岁 |
|------|-------|-------|-------|----------|

| 休息间隔 | 2分钟 | 3分钟 | 5分钟 | 5分钟 |

### 3. 测评问卷设计

#### 设计原则：题库随机抽取

- **题库规模**：设计80道题，覆盖7个评估维度
- **每次抽取**：每次测评随机抽取5-7道题
- **维度均衡**：确保每维度至少抽取1道题
- **年龄适配**：根据年龄段调整题目难度
- **防作弊**：同一维度题目打乱顺序抽取

#### 3.1 问卷维度分布

| 维度 | 题库数量 | 每次抽取 | 说明 |
|------|----------|----------|------|
| 持续注意力 | 12题 | 1道 | 评估长时间专注能力 |
| 选择性注意 | 12题 | 1道 | 评估干扰中专注能力 |
| 分配注意 | 10题 | 1道 | 评估多任务处理能力 |
| 转移注意 | 10题 | 1道 | 评估任务切换能力 |
| 工作记忆 | 12题 | 1道 | 评估信息存储加工能力 |
| 冲动控制 | 12题 | 1道 | 评估行为抑制能力 |
| 反应速度 | 12题 | 0-1道 | 可由游戏替代 |

**总计：80道题，每次抽取5-7道**

#### 5.2 百分位计算方法

百分位排名采用线性插值法计算：

```
percentile = (rank / total) × 100

其中：
- rank = 儿童得分在同龄组中的排名（从小到大）
- total = 同龄组总人数

对于边界值处理：
- P0 = 平均值 - 2×标准差
- P10 = 平均值 - 1.28×标准差
- P30 = 平均值 - 0.52×标准差
- P50 = 平均值（中位数）
- P70 = 平均值 + 0.52×标准差
- P90 = 平均值 + 1.28×标准差
- P100 = 平均值 + 2×标准差
```

#### 5.3 年龄标准化

由于不同年龄组的能力差异显著，所有测评结果都需要进行年龄标准化处理：

```javascript
// 年龄标准化公式
standardized_score = raw_score + (age_months - average_age_months) × adjustment_factor

// adjustment_factor 根据维度不同而不同
// 例如：工作记忆的adjustment_factor = 0.5（每月调整0.5分）
//       反应速度的adjustment_factor = 0.3（每月调整0.3分）
```

### 智能抽取算法

1. **维度均衡**：每个维度至少抽取1道题
2. **年龄适配**：根据年龄段选择对应题目
3. **随机打乱**：同维度题目打乱顺序
4. **动态数量**：根据年龄调整抽取数量
   - 4-5岁：抽取5-6题
   - 6-7岁：抽取5-6题
   - 8-9岁：抽取6-7题
   - 10-12岁：抽取6-7题
