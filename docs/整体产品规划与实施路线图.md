## 一、产品全景规划

### 1.1 产品愿景与使命

| 时间维度 | 愿景描述 |
|---------|---------|
| **3年愿景** | 成为国内最大的儿童专注力训练平台，覆盖100万+家庭，建立科学的专注力评估与训练标准 |
| **5年愿景** | 构建「AI+专注力」教育生态，成为家校协同的专注力提升基础设施，服务500万+儿童 |
| **核心使命** | 用科技的力量，让每个孩子都能获得科学、便捷、有效的专注力训练 |

**我们要解决的核心问题**：

| 痛点 | 现状 | 我们提供的解决方案 |
|------|------|-------------------|
| 效果焦虑 | 65%家长不知道训练是否有效 | 三层效果可视化体系（即时/周/月报告） |
| 时间成本高 | 线下机构每周接送2-3次 | 每天10分钟，线上完成 |
| 专业性不足 | 现有App多为小游戏堆砌 | 基于神经科学的9款专业训练游戏 |
| 价格门槛高 | 线下机构年费1.5-5万 | App价格仅为线下的1/20-1/50 |
| 孩子抗拒 | 训练枯燥像做作业 | 游戏化设计+AI情感陪伴 |

### 1.3 核心价值主张

**对孩子**：
- 🎮 **好玩**：游戏化的训练方式，让专注力训练变成有趣的游戏体验
- 🏆 **成就感**：即时反馈、徽章系统、排行榜，让孩子每次训练都有收获感
- 🤝 **陪伴**：AI小助手24小时在线，成为孩子成长路上的好伙伴

**对家长**：
- 📊 **看得见**：三层效果可视化体系，让家长每天都能看到孩子的进步
- 📚 **学得到**：家长学院提供科学的育儿知识，帮助家长成为专注力训练专家
- 💰 **值得**：199元/年起，每天不到6毛钱，比一节线下课还便宜

**对学校**：
- 📋 **减负担**：提供全校专注力普测系统，减轻教师观察负担
- 📈 **可追踪**：班级学情分析看板，帮助老师了解每个学生的发展状况
- 🔗 **可协同**：家校互通模块，打通家庭与学校的沟通壁垒

### 1.4 目标用户画像

#### 3-4岁段（幼儿园小班-中班）

| 维度 | 用户画像 |
|------|---------|
| **核心特征** | 语言表达快速发展，好奇心强，注意力以无意注意为主 |
| **家长画像** | 25-32岁新手父母，孩子刚入园，关注入园适应问题 |
| **核心痛点** | 孩子坐不住、容易被干扰、难以等待 |
| **训练目标** | 建立基础注意力习惯，提升持续关注时间至5-8分钟 |
| **付费意愿** | 199元/年（尝试型） |
| **内容偏好** | 简单指令执行、颜色形状识别、听觉专注训练 |

#### 5-6岁段（幼儿园大班-幼小衔接）

| 维度 | 用户画像 |
|------|---------|
| **核心特征** | 自我控制能力增强，开始形成有意注意，学业压力初现 |
| **家长画像** | 28-35岁，孩子即将升小学，焦虑幼小衔接问题 |
| **核心痛点** | 写作业分心、坐不住、做事磨蹭 |
| **训练目标** | 提升有意注意能力，建立任务坚持性，持续专注10-15分钟 |
| **付费意愿** | 199-399元/年（效果导向型） |
| **内容偏好** | 舒尔特方格、迷宫挑战、听觉记忆等与学业相关训练 |

#### 7-8岁段（小学低年级）

| 维度 | 用户画像 |
|------|---------|
| **核心特征** | 执行功能快速发展，学业任务繁重，专注力问题直接影响成绩 |
| **家长画像** | 30-40岁，孩子进入小学，深刻感受到专注力对学习的影响 |
| **核心痛点** | 写作业拖拉、考试粗心、上课走神、成绩不稳定 |
| **训练目标** | 提升执行控制能力，改善学业表现，持续专注15-25分钟 |
| **付费意愿** | 399元/年（甚至愿意为增值服务付费） |
| **内容偏好** | 工作记忆训练、复杂任务切换、抗干扰训练 |

### 1.5 产品Slogan与品牌故事

**产品Slogan**：

> **"每天10分钟，专注伴成长"**
> 
> *或：*
> 
> **"科学练专注，天天有进步"**

**品牌故事（简约版）**：

每个家长都希望孩子上课认真听讲、作业高效完成。但现实中，52%的家长因孩子写作业注意力不集中而崩溃，38.5%的学龄儿童专注力不达标。

我们相信：专注力不是天生的，而是可以训练的。

「专注星球」集结了神经科学、儿童心理学、游戏设计等多个领域的专家，基于50+年认知科学研究成果，为4-12岁儿童打造了一套科学、有趣、有效的专注力训练系统。每天10分钟，让每个孩子都能拥有专注的力量。
