## 二、AI专注力智能检测技术方案

### 2.1 行为数据分析体系

#### 核心指标定义

| 指标 | 计算方法 | 专注力含义 |
|-----|---------|-----------|
| **平均反应时（ART）** | 正确答题时间求均值 | 越短表示加工速度越快 |
| **反应时标准差（SDRT）** | 反应时离散程度 | 越大表示注意力波动越大 |
| **错误率（ER）** | 错误次数/总题数 | 反映注意力的持久性 |
| **漏答率（MR）** | 漏答题数/总题数 | 反映警觉性 |
| **虚报率（FAR）** | 错误按键/有效试次 | 反映冲动控制 |
| **波动指数（FI）** | 滑动窗口反应时变异 | 识别注意力漂移 |

#### 算法实现

```python
# 注意力评分算法（伪代码）
def calculate_attention_score(game_data):
    # 标准化各指标
    art_score = normalize(art, game_type_baseline)
    sdrn_score = normalize(1/sdrt)  # 越小越好
    er_score = normalize(1/er)     # 越低越好
    fi_score = normalize(1/fi)      # 越稳定越好
    
    # 加权求和
    attention_score = (
        art_score * 0.3 + 
        sdrn_score * 0.25 + 
        er_score * 0.25 + 
        fi_score * 0.2
    ) * 100
    
    return attention_score
```

#### 注意力波动识别

- **滑动窗口法**：每30秒计算一次窗口内反应时标准差
- **趋势检测**：识别持续下降或周期性波动
- **异常点标记**：标记明显偏离基线的试次

### 2.3 实时注意力评分算法

```python
# 实时贝叶斯注意力评分
class RealTimeAttentionScorer:
    def __init__(self):
        self.prior_mean = 70  # 先验均值
        self.prior_std = 15   # 先验标准差
        
    def update(self, current_score, observation_noise=5):
        """
        贝叶斯更新注意力评分
        current_score: 本次游戏表现评分（0-100）
        observation_noise: 观测噪声
        """
        # 后验均值
        posterior_var = 1 / (1/self.prior_std**2 + 1/observation_noise**2)
        posterior_mean = posterior_var * (
            self.prior_mean/self.prior_std**2 + 
            current_score/observation_noise**2
        )
        
        self.prior_mean = posterior_mean
        self.prior_std = np.sqrt(posterior_var)
        
        return self.prior_mean
    
    def get_confidence_interval(self, confidence=0.95):
        z = 1.96 if confidence == 0.95 else 2.576
        return (
            self.prior_mean - z * self.prior_std,
            self.prior_mean + z * self.prior_std
        )
```

**评分更新频率**：
- 游戏内：每秒更新瞬时评分
- 任务结束：计算综合评分
- 每日汇总：家庭训练日报评分

### 3.3 自适应难度调整算法

#### ZPD（最近发展区）实现

```python
class AdaptiveDifficulty:
    def __init__(self, target_accuracy=0.75):
        self.target_accuracy = target_accuracy  # 目标正确率75%
        self.difficulty_levels = [1, 2, 3, 4, 5]  # 5档难度
        
    def adjust(self, current_level, recent_accuracy):
        """
        根据最近表现调整难度
        """
        if recent_accuracy > 0.85:
            # 正确率过高，提升难度
            new_level = min(current_level + 1, 5)
            reason = "表现优秀，挑战更高难度"
        elif recent_accuracy < 0.60:
            # 正确率过低，降低难度
            new_level = max(current_level - 1, 1)
            reason = "需要巩固基础，逐步提升"
        else:
            # 正确率适中，保持难度
            new_level = current_level
            reason = "难度合适，继续保持"
            
        return new_level, reason
    
    def get_task_variants(self, base_task, level):
        """
        根据难度生成任务变体
        """
        params = {
            1: {'count': 9, 'time_limit': 60},   # 简单
            2: {'count': 12, 'time_limit': 50},
            3: {'count': 16, 'time_limit': 45},   # 适中
            4: {'count': 20, 'time_limit': 40},
            5: {'count': 25, 'time_limit': 35},   # 困难
        }
        return params[level]
```

#### 多维度难度参数

| 维度 | 难度1 | 难度2 | 难度3 | 难度4 | 难度5 |
|-----|-------|-------|-------|-------|-------|
| 刺激数量 | 9个 | 12个 | 16个 | 20个 | 25个 |
| 时间限制 | 60s | 50s | 45s | 40s | 35s |
| 干扰项数量 | 0 | 2 | 5 | 8 | 12 |
| 规则复杂度 | 单规则 | 双规则 | 变规则 | 混合 | 高阶 |

class FrustrationProtection:
    def __init__(self):
        self.consecutive_failures = 0
        self.frustration_threshold = 3
        
    def on_failure(self, current_difficulty):
        self.consecutive_failures += 1
        
        if self.consecutive_failures >= self.frustration_threshold:

        return None

            }

```python
class SchulteTableGenerator:
    def __init__(self):
        self.grid_sizes = {1: 3, 2: 4, 3: 5, 4: 6, 5: 7}
        
    def generate(self, difficulty, count=1):
        """
        生成舒尔特方格
        difficulty: 1-5
        count: 生成数量
        """
        grid_size = self.grid_sizes[difficulty]
        total_cells = grid_size * grid_size
        
        results = []
        for _ in range(count):
            # 生成随机序列
            numbers = list(range(1, total_cells + 1))
            random.shuffle(numbers)
            
            # 转换为矩阵
            matrix = [
                numbers[i*grid_size:(i+1)*grid_size] 
                for i in range(grid_size)
            ]
            
            results.append({
                'difficulty': difficulty,
                'grid_size': grid_size,
                'numbers': matrix,
                'time_limit': self._get_time_limit(grid_size),
                'expected_time': self._get_expected_time(grid_size)
            })
        
        return results
    
    def _get_time_limit(self, grid_size):
        """根据格子数量设定时间限制"""
        limits = {3: 30, 4: 45, 5: 60, 6: 90, 7: 120}
        return limits.get(grid_size, 60)
    
    def _get_expected_time(self, grid_size):
        """根据格子数量设定期望完成时间"""
        expectations = {3: 15, 4: 30, 5: 45, 6: 70, 7: 100}
        return expectations.get(grid_size, 45)
```

        )

```python
class ContentSafetyFilter:
    def __init__(self):
        self.banned_keywords = [
            '暴力', '色情', '政治敏感', 
            '赌博', '毒品', '自残', '自杀'
        ]
        
        self.child_unsafe_patterns = [
            r'.*伤害.*孩子',
            r'.*死亡.*游戏',
            r'.*危险.*挑战'
        ]
        
        self.appropriate_keywords = [
            '鼓励', '支持', '成长', '进步',
            '努力', '坚持', '勇敢', '开心'
        ]
    
    def check(self, content):
        """
        检查内容安全性
        """
        # 关键词过滤
        for keyword in self.banned_keywords:
            if keyword in content:
                return {'safe': False, 'reason': f'包含敏感词：{keyword}'}
        
        # 模式匹配
        for pattern in self.child_unsafe_patterns:
            if re.match(pattern, content):
                return {'safe': False, 'reason': '内容可能不适合儿童'}
        
        # 适龄检查
        appropriateness = self.check_appropriateness(content)
        if appropriateness < 0.5:
            return {'safe': False, 'reason': '内容可能不适合目标年龄段'}
        
        return {'safe': True, 'score': appropriateness}
```

#### 行业痛点

1. **推荐同质化**：多数产品使用固定训练计划

4. **内容枯竭**：任务种类有限，孩子容易厌倦
