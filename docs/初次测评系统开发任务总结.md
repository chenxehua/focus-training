## 一、项目概述

### 1.1 项目目标
为4-12岁儿童提供科学、简短（5-10分钟）的专注力评估系统，通过问卷+游戏的方式全面评估儿童7个维度的专注力发展水平。

### 1.2 核心功能
1. **智能问卷系统**: 从80题库中随机抽取5-7题，确保7维度全覆盖
2. **年龄适配游戏**: 5款游戏根据年龄自动调整难度
3. **科学评分系统**: 基于百分位常模的标准化评分
4. **可视化报告**: 雷达图展示7维度能力分布
