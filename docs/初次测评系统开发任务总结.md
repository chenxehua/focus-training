## 一、系统设计目标

### 核心功能
- 为4-12岁儿童提供科学、便捷的注意力评估
- 5-10分钟快速测评体验
- 80道题库，每次随机抽取5-7道题
- 5款分龄适配的游戏测评
- 即时生成个性化报告

### 7大评估维度
1. **持续注意力** (Sustained Attention)
2. **选择性注意力** (Selective Attention)
3. **分配性注意力** (Divided Attention)
4. **转移注意力** (Attention Shifting)
5. **工作记忆** (Working Memory)
6. **冲动控制** (Impulse Control)
7. **反应速度** (Reaction Speed)

### 4个年龄分组
- **4-5岁**: 学龄前儿童
- **6-7岁**: 小学低年级
- **8-9岁**: 小学中年级
- **10-12岁**: 小学高年级

function calculatePercentile(score: number, norm: PercentileNorm): number {

  return Math.round(normalCDF(z) * 100)
}
